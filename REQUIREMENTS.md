# 学术辅导文案工作台 — 需求文档

## 项目概述

结合大模型与用户数据，生成闲鱼学术辅导类商品文案，含敏感词防检测，支持多版本风格变体。

---

## 一、核心功能

| 功能 | 说明 | 状态 |
|------|------|------|
| 文案生成引擎 | 模板引擎 + LLM 润色 + 敏感词防检测 | ✅ |
| Web 管理面板 | FastAPI 后端 + Vue 前端 | ✅ |
| 批量生成 | 多组关键词一次产出 | ✅ |
| 多 LLM 支持 | DeepSeek / 通义千问 / OpenAI 可切换 | ✅ |
| 闲鱼爬虫 | 爬取学术辅导类商品供竞品分析 | ✅ |
| 一键启动 | start.bat | ✅ |

---

## 二、文案策略

### 身份定位
- **身份**: 个人博士团队（北邮博士），非中介
- **语气**: 专业真诚不浮夸，技术细节密集
- **价格**: 不写价格，引导私聊
- **结构**: 技术方向列举 → 会议/期刊级别 → 服务内容 → 引导沟通

### 风格变体
| 版本 | 风格 | 说明 |
|------|------|------|
| 标准版 | comprehensive | 完整技术列举 + 全面介绍 |
| 简洁版 | concise_casual | 核心方向，口语化表达 |
| 案例版 | case_led | 以成果/交叉学科案例开头 |

### 发布平台
- 闲鱼（goofish.com）

---

## 三、防检测机制

### 敏感词替换（4 层规则）

**直接替换**:
| 风险词 | 替换为 |
|--------|--------|
| 代写 | 写作协助 |
| 枪手 | 学术助手 |
| 包过 | 全力协助 |
| 包中 | 争取中稿 |
| 保过 | 保障质量 |
| 包录用 | 协助投稿 |
| 100% | 尽心尽力 |

**上下文替换**:
| 风险短语 | 替换为 |
|----------|--------|
| 论文代写 | 论文写作指导 |
| 代发论文 | 协助投稿 |
| 写手 | 学术指导 |

**安全短语白名单**（不触发替换）:
诚信第一、质量第一、顶级期刊、顶级会议、国家级项目、国家重点 等

**绝对禁止词**（出现在文案中立即告警）:
代写、枪手、包过、包中、保过、必过、100%、包录用、论文买卖、代发、写手代写、代做毕业设计、代写代码

### 频率控制
- 闲鱼爬取: 0.8 次/秒
- 页面间随机延迟 2±0.5 秒
- 参考工具: uutool.cn 敏感词检测

## 四、服务范围

### 研究方向
- 计算机视觉（目标检测、语义分割、图像生成、多模态融合等）
- 深度学习（YOLO系列、Mamba、Transformer、模型改进、消融实验等）
- 自然语言处理（LLM、RAG、CoT、Agent、情感分析等）
- 多模态 / 时序数据 / 具身智能

### 会议与期刊
- CCF A/B/C 类会议（AAAI、IJCAI、ICML、NeurIPS、CVPR 等 25+ 个）
- SCI 1-4 区 / 中文核心 / EI 会议

### 交叉领域
医学×AI、农业×AI、地理×AI、教育×AI

### 服务类型
- 提供思路 idea
- 内容优化、打磨 idea
- rebuttal 指导
- 全程辅导（实验+写作+投稿）
- 一对一腾讯会议讲解
- 算法改进优化 / 环境配置部署
- 论文润色与表达优化

### 目标受众
研一科研入门、研二小文章没思路、研三大文章缺创新/工作量、保研留学申博

## 七、技术栈

| 层 | 技术 |
|----|------|
| 后端 | Python 3.12+ / FastAPI / uvicorn |
| 前端 | Vue 3 + TypeScript (web/frontend/) |

## 八、API 端点

|------|------|------|
| POST | /api/generate | 生成文案 |

| GET | /api/stats | 统计数据 |
| POST | /api/export | 导出 TXT/JSON |

## 九、启动方式

```bash
# 一键启动
双击 start.bat

# 或手动启动后端
cd copy-generator
uv sync
uv run python -m uvicorn web.server:app --host 127.0.0.1 --port 8765

# 前端开发模式
cd web/frontend
npm install
npm run dev
```
